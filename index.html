This is a 2D cyclic cellular automaton simulator. Each cell has a color and if it is next to a cell with the color sequential to it, it will become that color in the next generation. Depending on the size it can take up to ~400 generations before stability occurs.

Functionality is not guaranteed beyond recommended intervals.

Size(20-400)
NumColors(2-15)
Interval between generations(ms)(100+)
Directions to check neighbors of
Top Left Top Top Right
Left Right
Bottom Left Bottom Bottom Right
Min number of adjacent neighbors in the right color required to change a cell(1-2)
